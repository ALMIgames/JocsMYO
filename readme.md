#Installation (without Homestead)
Clone this git repository into your local machine.

Run
```bash
composer install
cd /your/local/repository
php artisan serve
```

Go to
```bash
localhost:8000
```

In case of getting an error, un
```bash
cp .env.example .env
php artisan key:generate
```
and try again.
